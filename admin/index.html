⚠️ Erro ao carregar o CMS

O Netlify CMS precisa ser acessado através de um servidor HTTP local. Use um dos seguintes comandos:

  • Python 3: python -m http.server 8000
  • Node.js: npx http-server
  • PHP: php -S localhost:8000

Depois acesse: http://localhost:8000/admin/
